## 🚀 Sobre o projeto

O **SV AI** é uma ia desenvolvida em Python.

O objetivo do projeto é desenvolver, do zero, uma biblioteca educacional de Inteligência Artificial em Python. O foco é ensinar, de forma simples e prática, como funcionam neurônios artificiais, redes neurais e algoritmos de aprendizado.

---

## ⚙️ Status das funcionalidades

### 🟢 Implementadas
- Primeiro neurônio artificial
- Soma ponderada
- Bias
- Função de ativação simples (degrau)

### 🟡 Em desenvolvimento
- Organização da biblioteca

### 🔴 Planejadas
- Classe Neuronio
- Camadas neurais
- Rede neural
- Treinamento
- Backpropagation
- Salvamento de pesos
- Interface Web
